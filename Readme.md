
# is

  A full set of static methods to do type checking

## Installation

  Install with [component(1)](http://component.io):

    $ component install debianw/is

## API

  * isEmail()
  * isArray()
  * isEmpty()
  * isDate()
  * isString()
  * isBoolean()
  * isObject()
  * isSimpleObject()
  * isPrimitive()
  * isFunction()
  * isNumber()
  * isNumeric()
  * isElement()
  * isTextNode()
  * isDefined()
  * isIterable()
